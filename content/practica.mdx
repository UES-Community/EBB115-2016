## Ejercicio 1: Aplicación básica

Comenzamos con un ejercicio fundamental que integra los conceptos estudiados en la sección anterior. Lee el código, analiza su estructura y responde las preguntas al final.

```javascript title="ejercicio-1.js"
// Sistema de gestión básico
function crearRegistro(id, nombre, valor) {
  return {
    id,
    nombre,
    valor,
    timestamp: new Date().toISOString(),
    toString() {
      return `[${this.id}] ${this.nombre}: ${this.valor}`;
    }
  };
}

// Colección de registros
const registros = [];

function agregarRegistro(nombre, valor) {
  const id = registros.length + 1;
  const registro = crearRegistro(id, nombre, valor);
  registros.push(registro);
  return registro;
}

// Uso del sistema
agregarRegistro("Temperatura", 36.5);
agregarRegistro("Presión", 1013);
agregarRegistro("Humedad", 78);

registros.forEach(r => console.log(r.toString()));
```

<Callout type="info" title="Preguntas de análisis">
  1. ¿Qué patrón de diseño se puede identificar en el código anterior?
  2. ¿Cómo mejorarías la función `crearRegistro` para hacerla más robusta?
  3. ¿Qué validaciones agregarías?
</Callout>

## Ejercicio 3: Desafío de implementación

Implementa la función `filtrarYOrdenar` siguiendo las especificaciones:

```typescript title="desafio.ts"
interface Item {
  id: number;
  nombre: string;
  categoria: string;
  prioridad: "alta" | "media" | "baja";
}

// TODO: Implementa esta función
function filtrarYOrdenar(
  items: Item[],
  categoria: string,
  ordenarPor: keyof Item
): Item[] {
  // Tu implementación aquí
  return items
    .filter(item => item.categoria === categoria)
    .sort((a, b) => {
      if (a[ordenarPor] < b[ordenarPor]) return -1;
      if (a[ordenarPor] > b[ordenarPor]) return 1;
      return 0;
    });
}

// Datos de prueba
const items: Item[] = [
  { id: 1, nombre: "Tarea A", categoria: "desarrollo", prioridad: "alta" },
  { id: 2, nombre: "Tarea B", categoria: "diseño", prioridad: "media" },
  { id: 3, nombre: "Tarea C", categoria: "desarrollo", prioridad: "baja" },
  { id: 4, nombre: "Tarea D", categoria: "desarrollo", prioridad: "alta" },
];

const resultado = filtrarYOrdenar(items, "desarrollo", "prioridad");
console.log(resultado);
```

<Callout type="danger" title="Criterio de evaluación">
  La función debe manejar arrays vacíos sin lanzar errores. Asegúrate de probar con un array vacío como caso borde.
</Callout>
