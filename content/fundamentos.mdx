## Conceptos base

Los fundamentos de esta materia se construyen sobre tres pilares conceptuales que debes dominar antes de avanzar a los temas más complejos.

### Pilar 1: Definiciones esenciales

Todo marco teórico comienza con definiciones precisas. En EBB115-2016 trabajamos con un vocabulario técnico riguroso que garantiza la comunicación efectiva entre pares.

```python title="conceptos.py"
# Representación conceptual básica
class Concepto:
    def __init__(self, nombre, definicion):
        self.nombre = nombre
        self.definicion = definicion
    
    def describir(self):
        return f"{self.nombre}: {self.definicion}"

# Instanciando conceptos clave
c1 = Concepto("Abstracción", "Simplificar la complejidad ocultando detalles")
c2 = Concepto("Modularidad", "Dividir el sistema en componentes independientes")

print(c1.describir())
print(c2.describir())
```

### Pilar 2: Estructura y organización

La organización del conocimiento sigue un modelo jerárquico:

| Nivel | Descripción | Ejemplo |
|-------|-------------|---------|
| Macro | Visión general | Arquitectura del sistema |
| Meso | Componentes principales | Módulos y subsistemas |
| Micro | Detalles de implementación | Funciones y clases |

<Callout type="warning" title="Importante">
  No saltes niveles de comprensión. Cada nivel construye sobre el anterior. Dominar el nivel macro antes de estudiar el micro es clave para una comprensión sólida.
</Callout>
